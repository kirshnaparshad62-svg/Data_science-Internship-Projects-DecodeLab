# Data science Internship-Projects 
## Overview

This repository contains the projects I completed during my Data Science Internship. Throughout the internship, I worked on real-world datasets and followed practical data science workflows, starting from data preprocessing and feature engineering to building machine learning models and extracting business insights.

Each project focuses on a different stage of the data science lifecycle, helping me strengthen both analytical thinking and technical implementation using Python and the Data Science ecosystem.

---

## Internship Projects

### Project 1 — Advanced EDA & Feature Engineering

The internship began with building a strong foundation in data preprocessing. This project focused on transforming raw transactional data into a clean, structured dataset suitable for machine learning.

Key tasks included:

* Data cleaning and preprocessing
* Handling missing values
* Outlier detection using IQR and Z-Score
* Feature engineering
* Categorical encoding
* Feature scaling
* Preparing a machine learning-ready dataset

This project emphasized the importance of data quality before model development.

---

### Project 2 — Fraud Detection Using Machine Learning

The second project introduced supervised machine learning through an end-to-end fraud detection pipeline for e-commerce transactions.

The workflow included:

* Exploratory Data Analysis (EDA)
* Rule-based target label engineering
* Data preprocessing
* Handling class imbalance using SMOTE
* Feature scaling and encoding
* Training Logistic Regression and Random Forest models
* Hyperparameter tuning with GridSearchCV
* Model evaluation using multiple performance metrics

This project strengthened my understanding of building reliable and leak-free machine learning pipelines.

---

### Project 3 — Customer Segmentation Using Unsupervised Learning

The final project explored customer behavior using unsupervised machine learning techniques.

The project involved:

* Data preprocessing
* RFM and behavioral feature engineering
* Dimensionality reduction using PCA
* Customer segmentation with K-Means Clustering
* Cluster evaluation using Elbow and Silhouette methods
* Business interpretation of customer segments

This project demonstrated how machine learning can uncover meaningful patterns from unlabeled data to support business decision-making.
